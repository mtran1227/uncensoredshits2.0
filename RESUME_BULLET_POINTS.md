# Resume Bullet Points - Full Stack Development Project

• Built full-stack web app (React, Node.js, Express, MongoDB) with JWT authentication, bcrypt password hashing, and 30+ RESTful API endpoints supporting 65+ locations

• Integrated Google Maps API with MongoDB geospatial queries (2dsphere indexes) for real-time location-based searches and interactive map visualizations

• Developed responsive React frontend with 12 reusable components, React Router for client-side routing, and Tailwind CSS for modern UI design

• Designed 4 Mongoose data models with relational schemas (Users, Bathrooms, Ratings, VisitedBathrooms) and automated seeding scripts for 65+ campus locations

• Implemented multi-criteria rating system, social features (friend connections, bucket lists), and protected route middleware securing user sessions across the application
